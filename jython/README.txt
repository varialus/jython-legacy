Welcome to Jython 2.5.1rc2
==========================

This is the second release candidate of the 2.5.1 version of Jython. This
release fixes a number of bugs, including some major errors when using
coroutines and when using relative imports. Please see the NEWS file for
detailed release notes.

The release was compiled on Mac OS X with JDK 5 and requires JDK 5 to run.

Please try this out and report any bugs at http://bugs.jython.org.
