Welcome to Jython 2.5.2 Beta1
=============================

This is the first beta release of the 2.5.2 version of Jython. This release
fixes a number of bugs, including some Java integration problems, several modjy
bugs, a number of Python compatibility bugs, and more. See the NEWS file for
more details.

The release was compiled on Mac OS X with JDK 5 and requires JDK 5 to run.

Please try this out and report any bugs at http://bugs.jython.org.
