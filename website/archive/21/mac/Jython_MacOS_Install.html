How to install Jython on Macintosh

Prerequists:
You need
ftp://ftp.apple.com/developer/Development_Kits/MRJ_SDK_2.2_Install.sit.bin
I got it via - don't choose  MRJ 2.2.3 (because it has _no batteries includes_)
http://developer.apple.com/java/download.html

Install it.
The JBindery is the most useful application for installing Jython.

Get the Jython.class from
http://jython.sourceforge.net/

Install

Drag and drop jython-20a3.class on the JBindery Icon:

Choose Run

Now the install process should start:

At the first step you have to choose, which pakets will be installed

The System is going to be inspected

The install process is running

After installing, you'll find some new created folders

Next, the jython.jar - file has to be used to be started.

Here you have to put some data into JBindery:
at first the class name has to be choosen:
org.python.util.jython

Then the classpath has to be adjusted, so it includes the jython.jar (I added the Lib-Folder to), you may put any elements you need on this List.

Properties have to be set: this is the location, where your Jython-Home is located.

After all this little steps, Save Settings and then Run

The jar-Files are (the first time you start it up) processed, and the Jython-Interpreter is started:
