
About these documents
*********************

These documents are generated from reStructuredText sources by
*Sphinx*, a document processor specifically written for the Python
documentation.

Development of the documentation and its toolchain takes place on the
docs@python.org mailing list. 

Many thanks go to:

* Fred L. Drake, Jr., the creator of the original Python documentation
  toolset and writer of much of the content;

* the Docutils project for creating reStructuredText and the Docutils
  suite;

* Fredrik Lundh for his Alternative Python Reference project from
  which Sphinx got many good ideas.

See *Reporting Bugs in Python* for information how to report bugs in
this documentation, or Python itself.
